Observaciones:
El sistema permite registrar personas, registrar propiedades, crear publicaciones asociadas a propiedades disponibles y gestionar contratos de alquiler asociados a una propiedad y a un inquilino
La entidad Persona se utiliza para representar tanto propietarios como inquilinos
El archivo script_inicial.sql permite inicializar la base de datos con datos de prueba para verificar el funcionamiento de personas, propiedades, publicaciones y contratos
